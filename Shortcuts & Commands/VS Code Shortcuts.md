# VS Code Shortcuts

### General

|Operation|Command|
|---|---|
|open command palette|CMD + SHIFT + P|
|open file finder|CMD + P|

### Editing

|Operation|Command|
|---|---|
|delete line|SHIFT + CMD + K|
|move line down / up|OPT + DOWN / UP|


### Navigation

|Operation|Command|
|---|---|
|switch between tabs|CMD + OPTION + ARROW|
